# ORIA — LE MANDALA VIVANT
## *Spécifications complètes · Version 1.0*

---

## 1. CONCEPT ET VISION

Le Mandala Vivant est l'élément central et distinctif de l'application ORIA.

Il n'est pas un graphique de progression. Il n'est pas un dashboard de métriques.

Le Mandala Vivant est une **représentation symbolique et vivante de la transformation d'un être humain**.

**Ce qu'il exprime :**
- La totalité du parcours ORIA en une image
- Les couches de transformation traversées
- Ce qui est déjà intégré vs ce qui est encore en chemin
- L'unicité de chaque parcours (couleurs, symboles, densité)

**Ce qu'il n'est pas :**
- Un score
- Un classement
- Une comparaison
- Une évaluation de performance

---

### La métaphore du mandala

Dans les traditions contemplatives, le mandala représente la totalité — l'univers et l'être humain dans leur unité. Il se construit du centre vers la périphérie, ou de la périphérie vers le centre.

**Dans ORIA**, la progression est **de l'extérieur vers le centre** :

On commence à la périphérie — le contexte généalogique hérité, le monde extérieur imposé.

On chemine vers le centre — le Noyau, le Projet d'Âme, la vérité la plus intime.

À mesure que le participant avance, le Mandala s'illumine de l'extérieur vers l'intérieur — jusqu'à ce que le centre brille.

### Description détaillée de chaque couche

**Couche de Base (Phase 0 — Le Témoin)**
- Fond neutre, présent dès le début
- Représente la capacité d'observation — le sol du mandala
- Couleur par défaut : blanc ivoire / beige très clair
- Pas de validation visible — c'est la toile de fond

**Anneau 6 — Contexte Généalogique (Phase 1)**
- Premier anneau externe, le plus large
- Représente les racines, l'héritage, l'avant
- Symbole par défaut : racines / arbre / filament
- Couleur : choisie par le participant (dans une palette proposée)
- État validé : couleur pleine, légèrement lumineux
- État non validé : gris, texturé

**Anneau 5 — Projet-Sens (Phase 2)**
- Représente la mission héritée nommée et reconnue
- Symbole : fil / lacet / nœud (désigné par praticien)
- Couleur : choisie par le participant

**Espace Désert (Phase 2.5)**
- Visuellement : zone de transition entre Anneau 5 et Anneau 4
- Représentée par une bande légèrement texturée, granuleuse
- Couleur : nuance de gris-sable
- Pas de symbole — le vide lui-même est la représentation
- État : apparaît lorsque le praticien active la Phase 2.5

**Anneau 4 — Libération Systémique (Phase 3, première partie)**
- Représente ce qui a été posé, libéré, restitué
- Symbole : ouverture / envol / porte (désigné par praticien)
- Couleur : choisie par le participant

**Anneau 3 — Intégration (Phase 3, deuxième partie)**
- Représente l'ancrage somatique et corporel
- Symbole : corps / souffle / racine-ancre (désigné par praticien)
- Couleur : choisie par le participant

**Anneau 2 — Archétypes (Phase 5)**
- Représente la configuration archétypale identifiée
- Symbole : l'emoji/icône de l'Archétype principal
- Couleur : associée à l'Archétype (palette fixe par archétype)
- Particularité : peut afficher jusqu'à 3 archétypes en sections

**Anneau 1 — Contribution (Phase 4 + 6)**
- Représente la direction authentique, le don, la contribution
- Symbole : tiré du Portrait d'Âme (choisi par praticien)
- Couleur : choisie par le participant, plus lumineuse

**Centre — Noyau / Projet d'Âme (Phase 7)**
- Le cœur du mandala
- Représente le Noyau pleinement incarné
- Symbole : point lumineux + étoile ou soleil (choisi par praticien)
- Couleur : doré ou blanc lumineux — toujours lumineux, jamais terne
- Animation : léger pulsement lumineux lorsque complété

## 3. RÈGLES D'ÉVOLUTION DU MANDALA

### Principe général

Le Mandala ne peut évoluer que si :
1. Le praticien valide une phase
2. Le participant choisit sa couleur pour l'anneau débloqué

**L'évolution est un acte praticien + participant — pas un automatisme.**

---

### Séquence d'illumination

```
Phase 0 validée
    → Fond du mandala apparaît (légère luminosité)
    → Participant choisit couleur de base

Phase 1 validée
    → Anneau 6 s'illumine
    → Praticien attribue symbole (optionnel Phase 1)
    → Participant choisit couleur → appliquée à l'anneau
    → Animation : l'anneau "s'allume" progressivement (3 secondes)

Phase 2 validée
    → Anneau 5 s'illumine
    → Même séquence

Phase 2.5 activée (par praticien)
    → Espace désert apparaît entre Anneau 5 et Anneau 4
    → Texture granuleuse, animation légère de "brume"
    → Pas de couleur à choisir

Phase 3 validée
    → Espace désert se "dissout" (animation de clarification)
    → Anneaux 4 et 3 s'illuminent successivement
    → Deux couleurs à choisir (une par anneau)

Phase 5 validée + Archétypes attribués
    → Anneau 2 s'illumine
    → Symbole(s) d'Archétype(s) apparaissent dans l'anneau
    → Couleur de l'anneau = couleur associée à l'Archétype principal

Phase 4 + Phase 6 validées
    → Anneau 1 s'illumine
    → Symbole contribution assigné par praticien

Phase 7 validée
    → Centre s'illumine
    → Animation spéciale : illumination progressive depuis le centre
    → Tout le mandala "prend vie" — légère animation de résonance
    → Mandala complet — export possible en haute résolution
```

---

### Règles de personnalisation

**Ce que le participant choisit :**
- Couleur de chaque anneau (dans une palette proposée par phase)
- Intensité / saturation (curseur simple)
- Pas d'autres modifications visuelles

**Ce que le praticien choisit :**
- Symbole associé à chaque anneau (depuis une bibliothèque de symboles ORIA)
- Label optionnel pour chaque anneau (un mot, court)
- Archétypes → affichés dans l'Anneau 2

**Ce qui est fixe (non modifiable) :**
- Structure géométrique du mandala (concentrique)
- Proportion et taille des anneaux
- Séquence d'illumination
- Symbole et luminosité du centre

## 4. SPÉCIFICATIONS VISUELLES

### Rendu technique

```
Technologie : D3.js + SVG (rendu principal) + Canvas (export haute résolution)

Taille canvas de rendu :
  - Web (plein écran) : 800x800 px minimum
  - Mobile : 350x350 px
```

### Bibliothèque de symboles (pour le praticien)

Catégories disponibles :

**Racines & Origines :** arbre, racines, graine, branche, tronc, sève, humus

**Eau & Fluidité :** vague, rivière, source, marée, pluie, rosée, tourbillon

**Feu & Transformation :** flamme, braise, étincelle, forge, cendres, volcan, soleil

**Air & Inspiration :** vent, souffle, plume, aile, envol, brume, nuage

**Terre & Structure :** montagne, roche, cristal, grotte, fondation, forteresse

**Symboles ORIA :** spirale, noyau (point), voile, mandala, étoile, croissant

**Archétypes :** icônes spécifiques pour chacun des 12 Archétypes

### États visuels

```
Anneau non atteint :
  - Fond : gris clair transparent (opacity 0.15)
  - Bordure : tirets fins, gris
  - Pas de symbole

Anneau actif (phase en cours) :
  - Fond : couleur neutre légèrement saturée (opacity 0.3)
  - Bordure : continue, dorée légère
  - Animation : léger pulsement doux (2s)

Anneau validé :
  - Fond : couleur choisie (opacity 0.8)
  - Symbole visible
  - Légère lueur interne
  - Bordure : continue, couleur propre à l'anneau + légère brillance

Espace Désert :
  - Texture granuleuse / noise
  - Couleur : gris sable (#C8C0A8 à #E0D8C8)
  - Animation : légère oscillation horizontale

Centre complété :
  - Couleur or, opacity 1
  - Animation : pulsement lumineux régulier (3s)
  - Légère aura dorée autour du centre
```

## 5. INTERACTIONS UTILISATEUR

### Côté Participant

**Survol d'un anneau :**
→ Tooltip affichant le nom de la phase et la date de validation

**Clic sur un anneau validé :**
→ Panneau latéral : résumé de la phase, couleur, symbole, date de validation

**Clic sur anneau non atteint :**
→ "Cette étape sera débloquée par votre praticien en temps voulu."

**Bouton "Choisir ma couleur" (anneau actif) :**
→ Modale : palette de couleurs + aperçu en temps réel sur le mandala

**Bouton "Exporter" :**
→ Téléchargement PNG haute résolution (3000x3000px)

---

### Côté Praticien

**Vue du mandala participant :**
- Accès depuis la fiche participant
- Vue identique au participant + panneau de contrôle praticien

**Panneau praticien (côté droit) :**
- Liste des anneaux avec leur état
- Bouton "Valider cette phase" sur l'anneau actif
- Pour chaque anneau validé : sélecteur de symbole + champ label optionnel
- Attribution des Archétypes (Anneau 2)

## 6. EXPORT ET PARTAGE

### Export pour le Participant

**PNG haute résolution :**
- 3000x3000 px, fond blanc ou transparent
- Inclut les métadonnées : prénom, date de complétion, phases validées
- Téléchargement direct depuis l'app
- Filigrane discret "ORIA" en bas à droite

**Format carte numérique (optionnel V2) :**
- Format 1080x1080 (Instagram ready)
- Fond personnalisé avec nom du participant
- Tagline optionnelle du Portrait d'Âme

---

### Export pour le Dossier d'Âme (PDF)

**Page dédiée dans le Dossier d'Âme :**
- Mandala complet centré
- Légende des anneaux avec dates de validation
- Couleurs et symboles affichés
- Format A4 ou A3 selon préférence

---

### Partage (optionnel — contrôlé par le participant)

**Lien de partage :**
- Génère une page publique minimale avec le mandala (sans données sensibles)
- Durée du lien : 7 jours (renouvelable)
- Contenu visible : mandala image + prénom + "Parcours ORIA complété"
- Aucune donnée de parcours, phase, ou exercice visible

## 8. ANIMATION — SÉQUENCE D'ILLUMINATION

### Animation de validation (déclenchée par praticien)

```
1. Notification push au participant : "Votre praticien a validé une étape."
   
2. À la prochaine ouverture du mandala par le participant :
   a. Le mandala s'affiche dans son état précédent
   b. Pause de 1 seconde
   c. L'anneau nouvellement validé commence à s'illuminer
      - Fondu depuis opacity 0.15 → 0.85 en 2 secondes
      - Couleur : celle choisie par le participant (ou neutre si pas encore choisie)
      - Si symbole attribué : symbole apparaît en fondu à la fin (0.5s)
   d. Animation de "résonance" : léger pulsement unique sur tout le mandala (1s)
   e. Retour à l'état statique

3. Si participant n'a pas encore choisi de couleur :
   - L'anneau s'illumine en couleur neutre (gris-doré)
   - Un badge "Choisir ma couleur" apparaît sur l'anneau
   - À la sélection de couleur : transition fluide vers la couleur choisie (1s)

4. Animation finale (Phase 7 — Centre) :
   - Illumination progressive depuis le centre vers l'extérieur
   - Tous les anneaux "respirent" brièvement ensemble
   - Le centre pulse doucement en continu
   - Durée totale : 8 secondes
```
