# ORIA — Méthode de Transformation

> *Tu n'es pas venu guérir ton histoire. Tu es venu révéler ce que ton histoire tentait de protéger.*

ORIA est une méthode structurée de transformation personnelle et systémique, déployée via une plateforme SaaS dédiée aux praticiens certifiés et à leurs participants.

## Concepts clés

- **Le Noyau** — l'identité essentielle, avant les conditionnements
- **9 Voiles** — les couches de conditionnement qui recouvrent le Noyau
- **7 Forces ORIA** — les ressources transformatrices disponibles
- **12 Archétypes ORIA** — assignés par le praticien, jamais automatiquement
- **La Spirale ORIA** — le mouvement naturel de la transformation
- **Le Mandala Vivant** — représentation visuelle du parcours, 6 anneaux + centre
- **Le Dossier d'Âme** — synthèse PDF du parcours complet

## Les 4 Lois Fondatrices

1. Tout héritage cherche à se poursuivre
2. Ce qui n'est pas reconnu se répète
3. Ce qui est reconnu peut être transformé
4. Toute transformation appelle une contribution
